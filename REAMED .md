Descripción del Proyecto

Este proyecto implementa un simulador de gestión de venta de tiquetes y abordaje para un avión, utilizando el lenguaje C. El objetivo principal es aplicar los conceptos de **Gestión de Memoria Dinámica  

El sistema simula la sobreventa (10% sobre la capacidad) y respeta el orden estricto de venta (FIFO) para el abordaje, deteniéndose al alcanzar la capacidad máxima del avión.
